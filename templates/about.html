About The Website

Welcome to the steganollamaphy website!! This is a simple one-person website done as a final project for Harvard University's CS50 Introductory Computer Science 2023 course. The purpose of the project is to allow hiding information in images (practice called steganography), being able to store from text messages to links, including a link to an image within the image!!

How Does It Work?

The functioning of the steganography applied here is extremely simple compared to other algorithms used to hide information in images, but it is enough for the purposes of the site. Before understanding how it works, we need to understand how images are interpreted by computers, i.e., what their true general bit format is.

Basically, given a text message and an image, using the Pillow library, which adds support for opening and saving many different image formats, we convert the message into its respective unicode code into binary and save each digit in the last bit of the RGB pixel.
